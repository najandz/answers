Object Oriented JavaScript

Create an object model for a zoo in pure JavaScript. Use the principles of Object Oriented programming and inheritence to describe the various animals and where they live.

Note: Try to use inheritence in your demo. Eg: Start with an Animal class, and subclass to create the classes for each animal type.

When you are done, the following lines of code should produce the corresponding output below:


    zoo.pokeAnimals();

    // Lion says "Roar!", Goat says "Baaa!", Cat says "Meow", Horse says "Nee!"
